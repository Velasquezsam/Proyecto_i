Paso 1: Obtener el Repositorio

git clone https://github.com/Velasquezsam/Proyecto_i.git
cd Proyecto_i 

Paso 2: Crear y Activar un Entorno Virtual
Para evitar conflictos de versiones con librerías del sistema, 
es necesario aislar las dependencias creando un entorno virtual (venv). 
Abra su terminal en la raíz del proyecto y ejecute:

# Crear el entorno virtual llamado "venv"
python -m venv venv

#Dar permisos si no deja activar el entorno 
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# Activar el entorno virtual en Windows (PowerShell/CMD):
venv\Scripts\activate

# Activar el entorno virtual en macOS o Linux:
source venv/bin/activate

(Nota: El entorno estará activo cuando el prefijo (venv) 
aparezca al inicio de la línea de comandos en su terminal).

Paso 3: Instalar las Dependencias
Con el entorno virtual activado, instale las librerías exactas requeridas para el proyecto
 ejecutando el siguiente comando. 
Esto instalará versiones específicas para asegurar compatibilidad:

opc 1 : pip install -r requirements.txt
opc 2 : python -m pip install -r requirements.txt


Paso 4: Levantar el Servidor Web
Una vez que las dependencias y el modelo estén en su lugar, 
inicie la aplicación Flask: 

python app.py
